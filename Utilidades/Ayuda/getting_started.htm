Getting Started

DASHBOARD es una aplicación de consulta de datos realizada sobre la base de datos NorthWind. En la pantalla principal de la aplicación podemos encontrar diferentes secciones que nos ayudarán a llevar a cabo estas consultas.

Filtros

Esta sección se encuentra en la cabecera de la pantalla principal. En ella nos encotramos dos zonas diferenciadas:

Filtrado de clientes

Aquí puede realizar filtros sobre los clientes de la base de datos. Para ello puede hacerlo de dos formas diferentes:

  • Filtro directo por clientes.- Se despliega directamente el cuadro combinado de "Clientes", y se puede elegir un cliente de entre todos los que se encuentran en la base de datos.
  • Filttrado previo por países.- Primero se despliega el cuadro combinado de "Paises" seleccionando el pais deseado. Posteriormente se puede elegir en el cuadro de "Clientes" uno de los clientes correspondientes al pais elegido.

Búsqueda de clientes

En este apartado se pueden realizar búsquedas de cliente, para ello basta con hacer clic dentro del cuadro combinado de "Búsqueda de clientes" y comenzar a escribir. La aplicación dará a elegir entre los clientes que contengan el texto introducido.

Menú lateral

Esta sección se encuentra en la parte inquierda de la pantalla principal. Para activar cualquiera de los botones de está sección, es necesario haber seleccionado previamente un cliente en la zona de Filtrado de clientes.

  • General.- Muestra una pantalla donde se pueden ver las estadísticas del cliente seleccionado. Estas estadísticas son: Total de ventas, total de pedidos, descuento medio aplicado y un gráfico donde se muestran las ventas y las unidades compradas por el cliente.

  • Info Cliente.- Muestra una pantalla donde se pueden ver todos los datos del cliente seleccionado.

  • Pedidos.- Muestra los pedidos del cliente seleccionado y haciendo doble clic sobre uno de los pedidos se puede acceder al detalle de dicho pedido.

  • Reportes.- Muestra los posibles reportes que se pueden realizar:

    • Productos por categoría.- Listado de productos por cada categoría. Se debe introducir un número de categoría (Entre el 1 y el 8) para mostrar sus productos o el 0 para listar todas las categorías.
    • Pedidos en un periodo.-Listado de todos los pedidos cursados entre las dos fechas elegidas.
    • Productosmás vendidos.-Listado de los productos más vendidos entre las dos fechas elegidas.
